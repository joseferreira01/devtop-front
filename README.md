## 👍🏻 Como baixar

```bash

    // Clonar repositório
    $ git clone https://github.com/joseferreira01/teste-socin

    // Acessar diretório
    $ cd tests

    // Instalar dependências
    $ yarn

    // Iniciar projeto
    $ yarn start
```
